If you desire to display text as columns, then make display equal to block. Think blocks stacked on top of one another.*/ label { display: block; margin: 10px 0; /* Adds 10 pixels of space above and below each label element.*/ } /* If an element is nested within another element, such as a input nested in label, then the values with '%' will be with respect to their respective element. Ex. If label has width of 100px, and input has width of 50%, then input width = 50px.*/ input { margin: 10px 0; /* Adds 10 pixels space above and below each input and textarea element.*/ width: 100%; } /* When calling on values set by attributes, such as class="inline", inline must be written with a '.' before the name 'inline' */ .inline { width: unset; /* This overrides any previous width declarations */ } textarea { display: block; margin: 10px 0; height: 100px; width: 100%; }
